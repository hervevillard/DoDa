DODA ASSETS — PLACEHOLDER GUIDE
================================

Audio files needed (MP3 format, short clips):

  assets/audio/en/letters/   — a.mp3, b.mp3, c.mp3, d.mp3, e.mp3 ... (one per letter)
  assets/audio/rw/letters/   — a.mp3, b.mp3, c.mp3, cy.mp3, ny.mp3 ... (record with native Kinyarwanda speaker)
  assets/audio/en/words/     — apple.mp3, water.mp3, cat.mp3, dog.mp3, sun.mp3, ball.mp3, tree.mp3, house.mp3
  assets/audio/rw/words/     — pome.mp3, amazi.mp3, injangwe.mp3, imbwa.mp3, izuba.mp3, umupira.mp3, igiti.mp3, inzu.mp3
  assets/audio/rewards/      — star.mp3, success.mp3, encouragement.mp3

Image files needed (PNG, square, ~256x256):

  assets/images/words/       — apple.png, water.png, cat.png, dog.png, sun.png, ball.png, tree.png, house.png
                               tea.png, leaf.png, meat.png
  assets/images/ui/          — flag_en.png, flag_rw.png (small flag icons)
  assets/images/mascot/      — (optional) replace the drawn mascot with a real character sprite

Fonts:
  assets/fonts/Nunito-Regular.ttf
  assets/fonts/Nunito-Bold.ttf
  assets/fonts/Nunito-ExtraBold.ttf
  → Download from Google Fonts: https://fonts.google.com/specimen/Nunito

NOTE: The app works without these files — audio plays silently,
images show placeholder icons, and the mascot is drawn in code.
Add real assets when ready.
